One color language

Terrain type equals land resource identity.

This showcase uses the current generated terrain atlas and resource mask assets. The point is to collapse terrain and land-resource color into one semantic pair: the terrain surface gets the muted hue, while the yield plate and economy icon get the saturated version.

Rule

Remove mixed resource yields from terrain. Tiles can still vary by amount, slots, settlement value, or special rule, but the resource type is deterministic from the terrain type. This leaves one color scheme, not terrain colors competing with resource colors.

Political resources stay outside land colors

Influence and unrest are not terrain products, so they keep separate civic/status colors. They can appear in the economy bar, costs, tooltips, and event UI without competing with the four land pairs.
