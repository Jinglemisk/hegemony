## Hybrid rule
Icons earn their place only where the player scans a FIELD of many small like objects; text owns every surface read as a list. Specifically: (1) Board yield chits — pictogram + numeral, MANDATORY; terrain-by-fill-alone is the plain variant's one real failure (twin tans, colorblind risk). (2) Settlement markers — pictogram inside owner-colored ring (kind=glyph, owner=color); drop the letters. (3) Board pop chips — TEXT letters with counts (2C·1F·2S); person silhouettes are indistinguishable at chit size. (4) Board building slots — letter tiles (T/M) + dashed empties; no micro-buildings. (5) Verb bar — text-first; icons permitted as flavor but may never replace the word, and costs stay text. (6) Resource spine — icon + word + tabular numeral: the icons pay rent here solely because they repeat the board-chit vocabulary (leaf on board = leaf in spine). (7) Ledger per-city delta tables — icon column headers (same spine glyphs) beat WO/ST/FO letter pairs; all other ledger lines, warnings (DISCONTENT), chronicle, tooltips, and modals stay pure text with color doing state. (8) Rail tabs — text required, icon optional (only useful if the rail ever collapses). Invariant: one glyph = one resource everywhere; never icon-only without an adjacent numeral or word except settlement kind.

## Where stylization pays vs where it's free flavor

**Pays real reading time (keep):**
- **Board yield chits.** This is the entire argument. A pictogram beside the numeral turns "find food" into a pop-out search; fill-color-only fails between food-khaki and gold-mustard, degrades on uncalibrated monitors, and excludes colorblind players.
- **Settlement kind glyphs.** Temple/flag/castle at marker size are one-fixation kind reads; geometric shape + repeated letter is two.
- **Spine and ledger-table resource glyphs** — not for themselves, but because they close the loop with the board chits: one visual vocabulary from tile to total.

**Free flavor (harmless, keep only if cheap):** verb-bar icons, rail-tab icons, the season seal, the modal kylix. In every timed task the *word* carried the meaning; the glyph was ambient.

**Actively worse than plain (cut):** person-class silhouettes on board pop chips — at that size 2-citizen vs 2-slave silhouettes are identical black blobs, while plain's `2C·1F·2S` is unambiguous; micro-building glyphs in slot rows lose to T/M letter tiles.

## Where plainness costs

Scan speed only on the board (task 1's 3x penalty is disqualifying for an RTS-legibility bar). Charm: the number-only board reads as a heatmap spreadsheet, not a game — competent but austere. Everywhere else — verbs, spine deltas, ledger, warnings, tooltips, chronicle — plainness costs *nothing*, and twice (pop chips, slot tiles) it wins outright.

## Brandbook hybrid rule

> **Icons where the player scans a field; text where the player reads a list.**

| Surface | Ruling |
|---|---|
| Board yield chits | **Icon + numeral, mandatory** |
| Settlement markers | **Pictogram in owner-colored ring** (kind=glyph, owner=color, no letters) |
| Board pop chips | **Text**: `2C·1F·2S` style |
| Board slot rows | **Letter tiles** (T/M) + dashed empties |
| Verb bar | **Text-first**; icons optional, never replacing the word; costs always text |
| Resource spine | **Icon + word + tabular numeral** (glyphs must match board chits) |
| Ledger delta tables | **Icon column headers** (spine glyphs); all other ledger lines text |
| DISCONTENT / warnings | **Word + color**, no glyph needed |
| Rail tabs | **Text required**; icon optional (for future collapsed rail) |
| Chronicle, tooltips, modals | **Pure text**; modal seal is optional flavor — plain's "CIVIC ACTION" kicker is the more informative header |

**Invariants:** one glyph = one resource everywhere it appears; no icon ever appears without an adjacent numeral or word, except settlement kind, which earns the exemption by being the board's largest mark.

**Bottom line:** neither variant ships as-is. The board must be stylized; the chrome should stay essentially plain. The stylized variant wins the argument only where ink meets hexes — grant it that territory and nothing else.
